🚇 交通出行分析（核心方向）#

1️⃣ 共享单车租赁需求预测#

  • 基于时间序列与用户行为数据，分析不同时间段的租赁需求变化
  • 构建特征：天气、时间、节假日、温度等多维变量
  • 使用回归模型进行需求预测
  • 输出结果用于优化车辆调度与投放策略

👉 关键词：需求预测 / 时间序列 / 特征工程

2️⃣ 全球酒店预订行为分析#

  • 分析用户在不同国家/地区的酒店预订行为差异
  • 构建用户画像：价格敏感度 / 预订提前期 / 目的地偏好
  • 挖掘高价值用户群体特征
  • 为平台推荐系统提供数据支持

👉 关键词：用户画像 / 行为分析 / 推荐系统基础

3️⃣ 旅游景点热度数据分析#

  • 基于全球景点数据分析热门旅游趋势
  • 对比不同国家/城市的流量分布规律
  • 构建景点热度评分模型

👉 关键词：数据可视化 / 热度模型 / 区域分析

💰 金融数据分析（重点方向）#

4️⃣ 信用卡交易欺诈检测#

  • 使用统计方法 + 机器学习识别异常交易行为
  • 处理类别极度不平衡问题（Imbalanced Data）
  • 提升模型对欺诈行为的召回能力

👉 关键词：异常检测 / 风控 / 不平衡数据

5️⃣ 拍拍贷用户信用风险分析#

  • 分析用户还款能力与违约概率
  • 使用特征工程优化模型表现
  • 为信贷审批提供决策支持

👉 关键词：信用评分 / 风控模型 / 金融建模

🧠 数据分析思维#

我认为高级数据分析的核心不只是“算数据”，而是：

  • 理解业务逻辑，而不是只看表结构
  • 用数据验证假设，而不是堆指标
  • 从结果反推原因，而不是只做展示
  • 用统计思维降低决策不确定性

🎯 当前学习方向#
